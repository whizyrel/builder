Hi! This wasn't built to be extremely smart. It was just written for the purpose of ease in making the json file. Please be cautious as you use it (the verses could be messed up if order is not fullowed).

Order:

  1. Enter book title like Genesis...
  2. Chapter No then add button to add it.
  3. Paste verse and click to add it.
  4. When you are through with a chapter you could add another chapter with the respective button below and contine with adding verses to the newly added chapter in the usual way. When you are through (adding verses to the chapter), use the submit button to save it and the download button to download it.

Warning:

It is volatile! On refreshing this page you would have to start all over again! Please be careful
